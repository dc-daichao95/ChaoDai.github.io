更新博客文件

常规来说更新Hexo的博客只需要在修改之后，hexo clean;
个人觉得比较麻烦，简单写了一个shell和expect的脚本（也可以用crontab定时自动备份）

  • syn_blog.sh

    #!/bin/bash
# 基于hexo的博客同步到github page的脚本
blog_home="/home/dc/Workspace/blog"
script_ge="/home/dc/Workspace/script/hexo_generate.expect"
(cd $blog_home;hexo clean;hexo g; expect $script_ge)

  • hexo_generate.expect

    #!/bin/expect
set timeout 30
set username "XXX@XXX.com"
set passwd "XXXXXX"
spawn hexo d
  expect "Username for 'https://github.com':"
  send "$username\r"
  expect "Password for 'https://$username@github.com':"
  send "$passwd\r"
  interact
    将博客目录下的源文件备份到github（虽然现在github也不稳了）

    首先在github博客仓库下新建一个分支hexo_bak，然后git clone到本地，把.git文件夹放在博客目录下。

    查看远程分支
$ git branch -a
查看本地分支
$ git branch
切换分支
$ git checkout -b hexo_bak
添加和上传文件到本地仓库
$ git add .
$ git commit -m "infos"
上传本地仓库(本分支第一次上传)
$ git push --set-upstream origin hexo_bak
